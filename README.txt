##################################README###########################
1. Installed NodeJs setup
2. run node install --->>> for add required package in project
3. Chnage root folder path in file ./app.js
   - Chnage this line accordingly   -----   var rootpath = '/home/piyush/localserver';
4. run server using  ---->  node bin/www
5. Check browser on machine with url ------> localhost:5000
6. if error shown 'port is used', then chnage port to any other available port in file bin/www 
7. After succesfull run server , run browser on any local device with url <IP of server>:5000.
8. Enjoy Sharing locally :-)
